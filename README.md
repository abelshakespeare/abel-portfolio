### Run Locally

To run **Julia** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/julia-portfolio.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/julia-portfolio.git
```
